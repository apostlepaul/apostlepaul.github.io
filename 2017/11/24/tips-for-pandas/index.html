tips for pandas

发表于 2017-11-24 | 分类于 tech notes | 本文总阅读量 次

用pandas做少量数据分析还是比较方便的。
但是pandas用的不熟练，所以记录一些常用的tip作为积累。持续更新。

基础操作

  • 构造DataFrame
    data = pd.DataFrame(read_in_numpy_array(f, sep="\t"), columns=cols)
  • 按某个条件选择数据
    data = data[(data.c1 == "1") | (data.c1 == "2")]
  • 查看某些列
    res[col_names]
  • 保存文件
    data.to_csv('data.csv')

  • 数据合并，连接可以用merge, join, concat,函数原型分布是,这个博客讲的比较好。
    merge(left, right, how='inner', on=None, left_on=None, right_on=None, left_index=False, right_index=False, sort=True, suffixes=('_x', '_y'), copy=True, indicator=False)
    只能呢merge DataFrame。
    join(self, other, on=None, how='left', lsuffix='', rsuffix='',sort=False)
    concat(objs, axis=0, join='outer', join_axes=None, ignore_index=False, keys=None, levels=None, names=None, verify_integrity=False, copy=True)
    可以concat Series。

  • 调整显示出来的行
    with pd.option_context('display.max_rows', None, 'display.max_columns', 3): print(sort1[:50])

画图

  • 获取图的句柄
    ax=plt.gca()
  • 设置grid
    ax.grid(True)
  • 标签选择90度
    ax.set_xticklabels(tmp.date,rotation=90)
  • 设置轴范围
    plt.xlim(0,0.3) plt.ylim(0,0.3)

处理数据

  • 对某一列进行map处理
    data['c1'] = data.c1.apply(lambda x: x.split('%')[0])
  • 转化成数值
    data[cols[3:]] = data[cols[3:]].apply(pd.to_numeric)
  • 行增拼接数据
    new_res = pd.concat([new_res,tmp], ignore_index=True)
  • 按index join两个DataFrame
    a.join(b)
  • 排序
    data.sort_values(by=col_names, ascending=False)
    ``

    pivot table

  • 构造pivot table
    t = pd.pivot_table(data, index=index_names, values=value_names, aggfunc='sum')

  • 拼接两个table
    all_table = pd.merge(exp_table, dz_table, how="outer", left_index=True, right_index=True,suffixes= ['_exp','_dz'])
