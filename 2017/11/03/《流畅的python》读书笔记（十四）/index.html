第16章 协程

python中协程的核心关键字是yield。把yield视作控制流程的方式，会便于理解协程。
python2.5才把生成器进化成了协程。

1. 基本过程

简单地来理解，yield就是一个标志，执行到yield这一句时，程序会执行以下过程

  1. 把yield右边的返回。
  2. 然后程序就暂停了，处于“GEN_SUSPENDED”的状态。
  3. 对程序调用了next()或send()，才会往下一个yield运行。如果调用了send()，那么会把send的值赋给yield左边的变量。
  4. 所有的yield运行完了，会抛出StopIteration。

3. 终止和异常处理

调用.close()会使协程结束。
调用.throw(exc_type)，会使协程在yield的地方传入的异常exc_type, 如果这个异常被协程捕获了，协程处理，然后接着暂停，如果没有被捕获，那么协程会停止。
如果不论怎么停止，都要做一下清理工作，需要把协程定义体中的相关代码放到try/finally中。

4. 让协程返回值
