第14章 可迭代的对象、迭代器和生成器

1. 可迭代对象和迭代器

可迭代对象：使用iter()内置函数可以获取迭代器的对象，如果对象实现了能返回迭代器的__iter__()方法，那么对象就是迭代的。序列都可以迭代;实现了__getitem__()的方法也是可迭代对象。

迭代器：实现了无参数的__next__()方法，返回序列中的下一个元素，如果没有元素了，抛出StopIteration异常。实现了__iter__()方法。

标准的迭代器接口有两个方法：

  • __next__():返回下一个可用元素，如果没有元素了，抛出StopIteration异常。
  • __iter__():返回self，即自己这个迭代器。

可迭代对象一定不能是自身的迭代器，否则适用性会变差。既，可迭代对象应该实现__iter__()方法，而不应该实现__next__()方法。

2. 生成器函数

__iter__()返回的，可以是迭代器，也可以是生成器对象（生成器函数）。
只要函数体中有yield函数，该函数就是生成器函数，调用生成器函数时，会返回一个生成器对象。生成器函数是生成器工厂。
生成器函数还可以用生成表达式来实现。生成器表达式可以理解为列表推导的惰性版本。

3. 标准库中的生成器函数

再喊一遍口号：不重复造轮子！

过滤

  • itertools.takewhile(predicate, it): 就像while循环。
  • itertools.filterfalse(predicate, it):跟内置的filter相反。

    映射

  • itertools.accumulate(it, [func]):累加。
  • itertools.starmape([func], it):把*it传给func。

    合并

  • itertools.chain(it1,…,itN):串起来。
  • itertools.zip_longest(t1,…,tN,fillvalue=None):

    扩展元素

  • itertools.combinations(it, out_len): 把it的out_len个元素组合在一起，然后产出。
  • itertools.cycle(it):循环产出。
  • itertools.repeat(item, [times]):重复n次。常见用途：为map函数提供固定参数。
  • itertools.tee(it, n=2):产出的是n个单独的it迭代器，可以认为是复制了n个it迭代器。

iter()一个鲜为人知的用法：传入两个参数，第二个是哨负，迭代遇到哨符，会抛出StopIteration，停止迭代。
