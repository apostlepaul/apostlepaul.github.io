my usage of protobuf

发表于 2017-12-17 | 分类于 tech notes | 本文总阅读量 次

感觉protobuf是神器啊。google良心出品。

1. 基础用法

直接参考tutorial吧，写的很好，翻不了墙的还能看中文翻译版本。
简单来说，c++版本就是以下使用步骤：

  1. 定义proto文件；
  2. 用protoc编译proto文件，产出.pb.cc和.pb.h文件；
  3. 在需要使用的地方include .pb.h文件；
  4. 把proto里的message当做一个已经声明好的class用。有各种接口，getter, setter。

2. 序列化

有接口，序列化到string或者stream。

  • bool SerializeToString(string* output) const;
  • bool ParseFromString(const string& data);
  • bool SerializeToOstream(ostream* output) const;
  • bool ParseFromIstream(istream* input);

3. 遍历field

手写出来所有的field会很啥，可以使用描述符（Descriptor）和反射（Reflection）来得到字段的描述和字段的值。

  1. 得到message的Descriptor和Reflection.
    const Descriptor *des = m->GetDescriptor();
    const Reflection *ref = m->GetReflection();
  2. 通过Descriptor可以得到message的名字
    des->full_name()
  3. 通过Descriptor得到field的总数
    int fieldCount= des->field_count();
  4. 可以根据field进行for循环遍历
    const FieldDescriptor *field = des->field(i);
    field->name();
    field->type_name();

  5. 根据type可以进行类型的判断，从而通过Reflection得到相应的值。可以各种get, 接口都在官方文档。

  6. 如果field是repeated，可以根据FieldSize得到个数，然后用接口里的GetRepeatxxx的方法得到。

  7. 如果field是message，可以把message拷出来，递归调用。

以上都是跟stackoverflow的帖子学的。
