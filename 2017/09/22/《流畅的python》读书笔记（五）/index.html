第5章 一等函数

一定对象的定义：

  • 在运行时创建
  • 能赋值给变量或数据结构中的元素
  • 能作为参数传给函数
  • 能作为函数的返回结果
    python中的函数就是一等对象，也叫“一等函数”。

1.把函数视作对象

__doc___属性用于生成对象的帮助文博。可用help(func)来查看。
python中函数是可以作为一个参数传给另外一个函数的。

2.高阶函数

接收函数为参数，或者吧函数作为结果返回的函数，叫高阶函数。例如map函数就是高阶函数。

map/filter/reduce/apply是最常见的高阶函数。不过它们都有更好的替代品。

map和filter可以用生成器表达式来代替。

书上说reduce可以用sum代替，但是我觉得reduce的功能不局限于sum。

all()和any()也是归约函数，用来判断真值的。

4. 可调用对象

判断对象是否可调用，就是说，对象后面能否加括号，用callable(obj)。
可调用的对象包括：

  • 用户定义的函数
  • 内置函数
  • 内置方法
  • 方法
  • 类
  • 类的实例
  • 生成器函数

5. 用户定义的可调用类型

在对象中实现__call___，对象就是可调用的，所谓创建了函数类对象。

6. 函数内省

函数专有而用户定义的对象没有的属性包括：

  • annotations
  • call
  • closure
  • code
  • defaults
  • get
  • globals
  • kwdefaults
  • name
  • qualname

8. 获取关于参数的信息
